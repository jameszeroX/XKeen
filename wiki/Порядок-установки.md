Для установки XKeen требуется [роутер Keenetic](Routers-Info) с установленной средой [Entware](https://support.netcraze.ru/giga/nc-1012/ru/20980-installing-the-entware-repository-on-a-usb-drive.html)
Если планируете использовать XKeen в режиме TProxy, то в настройках доступа к веб-интерфейсу перенесите 443 порт на другой. Для режима Hybrid это не требуется

Выполните в ssh-консоли роутера следующие команды

```
opkg update && opkg install curl tar && cd /tmp
sh -c "$(curl -sSL https://raw.githubusercontent.com/jameszeroX/XKeen/main/install.sh)"
```
либо:
```bash
opkg update && opkg install curl tar && cd /tmp
sh -c "$(curl -sSL https://cdn.jsdelivr.net/gh/jameszeroX/XKeen@main/install.sh)"
```

Для создания конфигурационных файлов можете использовать различные генераторы в интернете, но лучше, разобравшись с документацией ядер, составить файлы вручную. Это убережет от многих возможных ошибок, которые допускают новички.

Для использования ядра Xray необходимо сконфигурировать два файла, которые находятся в директории /opt/etc/xray/config/

04_outbounds.json - файл с настройками подключения к серверу
05_routing.json - файл с настройками выборочной маршрутизации

По умолчанию XKeen с ядром Xray настроен на работу в режиме Hybrid. Если необходимо переключить его в режим чистого TProxy, замените так же следующий файл конфигурации

[03_inbounds.json](https://github.com/jameszeroX/XKeen/releases/latest/download/03_inbounds.json) - файл, переключающий режим на TProxy

Для ядра Mihomo необходимо сконфигурировать один файл config.yaml в директории /opt/etc/mihomo/ по инструкциями в интернете или используя генераторы

Известные генераторы конфигурационных файлов:
https://xray-routing-generator.netlify.app
https://zxc-rv.github.io/XKeen-UI/Outbound_Generator/
